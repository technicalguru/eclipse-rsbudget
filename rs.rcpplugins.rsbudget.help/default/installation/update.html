Online Update

RsBudget comes with an Online Update functionality. As there is no special notification, you will need to trigger it manually. The system will check the Update Site whether there is a new version available and ask you to confirm the update. The application will restart when the update was installed.

You can check the availability of an update also by going to the RsBudget Homepage.

Important! If you sit behind a proxy then it might be required to adjust the Network Settings.
