Configuration

The configuration will start when you first start RsBudget. Just double-click on the executable that you will find in the extracted folder.

The program will ask first which language you want to use for setup. Select your language and click OK to start the configuration. You should see a Welcome Screen in your chosen language.

Click "Next" to see the Database Configuration Page.

Database Configuration

RsBudget stores all your data in a database. The choice which database it uses is on you. This page asks you for details, especially which database product you chose and the access information (login, password) to it.

An important question is the "Use Full Encryption" checkbox. Full encryption means that all your data in the database will be encrypted so that other users of your computer cannot read it even though they get access to your database. You should keep in mind that this setting cannot be changed once the configuration completed.

Full encryption uses your application password (see next section) to encrypt and decrypt the data. As your application password is not stored anywhere, losing your password means that you lose all your data. So consider carefully whether you want switch on full encryption.

Click the "Test Connection" button to check whether your setting are correct. The program will inform you about any existing data and mismatches in your database settings.

Click "Next" to assign your application password. If you didn't test the database connection yet manually, the program will do so now and will not proceed if it detects any misconfiguration.

Setting an Application Password

The application password protects your data from any unauthorized users. It is not stored anywhere on your computer so you don't need to be afraid that anyone can ever steal it by stealing your computer or scanning your disk.

The program will ask you first each time it starts up to make sure that no unauthorized persons access your data. However, professional attackers could either see your data directly in the database (if you didn't select full encryption) or grab your password by spy software when you are typing it in. There is no 100 percent safety. ;-)

Type in your password twice so RsBudget can be sure that you entered the password correctly.

Click "Next" to setup your default bank account.

Default Bank Account

RsBudget requires each transaction to be connected with a bank account. This can either be your real bank account or any "virtual" that you make up for the program. There is currently no further meaning of that bank account. However, future extensions will be made available which can download transactions automatically from your bank so you save typing effort.

The Bank selection is currently based on German banks only. If you don't have a German bank account, just select any bank randomly and continue. As mentioned before, there is currently no further action behind this setting. A future update could enhance this to care for others than German banks.

Enter the name that you want to refer to for your account and the account number and owner as known by the bank.

Click "Next" to setup the default transaction category.

Default Transaction Category

Each transaction is associated with a category. You will be able to create your own categories later in the program. However, at the beginning at least one category is required that all transactions are assigned to by default. RsBudget proposes "Misc" as the name for it, but you can change it to whatever you feel appropriate.

Click "Next" to review your settings.

Setup Summary

The last screen shows again all settings that you have done in the previous screens. Review them carefully and press "Finish" to confirm them. Otherwise, click "Back" to go back to the setting page that you need to change.

RsBudget will now create all required database tables and initial data and, if successful, start up with the main screen. Congratulation! Your installation is ready to use.

Go to the Introduction page to learn the philosophy of how RsBudget manages and logs your transactions. The First Steps help page will show you how to begin and how to plan and manage your first budgets.
