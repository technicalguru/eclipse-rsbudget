Software Prerequisites

Before you can install RsBudget, you will need to install additional software on your computer. This chapter guides you through the process of preparing your computer.

Java Runtime Environment

RsBudget is based on Eclipse' Rich Client Platform (RCP) which is fully based on Java. However, Java is not part of the RsBudget installation package. That's why you will need a current Java Runtime Environment (JRE). Please, download and install at least a JRE 7 on your computer. RsBudget was developed and tested with Oracle's JRE, so it is recommended to use their product. Follow the instructions of your product choice to install the JRE.

Database Software

RsBudget saves all your information in SQL based database engine. Here is a list of choices.

  • HyperSQL (HSQLDB) - no further installation required
  • MySQL 5.x
  • Oracle - generally supported but not tested (see Other Databases)
  • DB2 - generally supported but not tested (see Other Databases)

There is no additional installation required if you opt for HyperSQL. The required software is packaged along with RsBudget.

Using HyperSQL

There are no additional steps to perform for HyperSQL. The software is already packaged with RsBudget. This choice is 2nd best only as HyperSQL is not tested thoroughly enough with RsBudget.

Using MySQL

MySQL 5.x shall be used only when you are an expert of this software. Install MySQL according to the MySQL installation instructions. Additionally, you will need to perform these steps:

  1. Create a database, e.g. rsbudget.
  2. Create a database user rsbudget (or any other name) and assign an appropriate password to it.
  3. Grant all privileges on the new database to the new database user. This shall especially include CREATE, ALTER and DROP TABLE privileges as well as SELECT, INSERT, UPDATE, DELETE privileges.

Remember your setup as RsBudget will ask for this information in the initial configuration wizard.

Other Databases

It is generally possible to use any database supporting JDBC and Hibernate. If you choose to use such a database for your RsBudget installation, you will need to perform additional steps. The current version, however, does not support them out-of-the-box. Therefore, this is recommended for experts in Java development, only. You will need to perform the following tasks:

  • Install your database product
  • Setup the database user
  • Create the database instance that holds RsBudget data
  • Create the database tables
  • Create initial data
  • Create the dbconfig.xml file in your home directory
  • Provide the JDBC driver via classpath to RsBudget

The best way to perform all these tasks is to setup RsBudget for MySQL first and then copy all database data to your product. Finally, you will need to change the dbconfig.xml file that RsBudget has created while performing the initial setup in your $HOME/.rsbudget directory.

A future version of RsBudget will support other database products.
