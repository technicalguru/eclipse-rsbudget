General Planning

This view enables the General Budget Planning.

Data Description

The view is divided into two separate sections. The upper part is the Summary Section. It shows basic figures of your General Budget Planning. The lower part shows the transactions planned for individual periods.

Summary Section

The section summarizes the plan so you can see your financial health.

Field Description
Monthly Income The total of all incoming transactions per month
Monthly Expenses The total of all outgoing transactions per month
Monthly Balance The profit/loss calculation of all transactions per month
Yearly Income The total of all incoming transactions per year
Yearly Expenses The total of all outgoing transactions per year
Yearly Balance The profit/loss calculation of all transactions per year
Average Monthly Income The average income in a month. The figure includes and levels all incoming transactions in a year.
Average Monthly Expenses The average expenses in a month. The figure includes and levels all outgoing transactions in a year.
Average Monthly Balance The average profit/loss calculation per month. The figure includes and levels all transactions in a year.

Transaction Table

The transaction table shows the planned transactions. A turquoise background identifies a budget. White background identifies single transactions. The following table describes the values displayed in each column.

Column Description
Cycle The periodicity of the transaction (monthly, quarterly, half-yearly, yearly)
Occurance The months this transaction will occur in (blank for monthly transactions).
Text The transaction text
Details Details and remarks of the transaction, e.g. origin or receiver (optional).
Budget The budget this transaction belongs to (optional).
Category The category this transaction belongs to.
Amount The amount this transaction is planned with. Positive numbers are incomes, negative numbers are expenses. The amount is colored according to your color preferences.

Tasks

Create a new transaction

Create a new transaction by

  • Right-click on the transaction table. Select "New Transaction" from the context menu.
  • Click the "New Transaction" button directly above the transaction table.
  • Click the "New Transaction" button from the view's toolbar.
  • Press the "Insert" key.

All methods will create a new transaction at the end of the table and scroll to the new row in order to make it visible. The editing cursor is placed within the Text field so you can start typing.

Modify a transaction

Double-click the value in the transaction table that you want to modify. Depending on the column, a text editor or combo box will appear so you can change the current value of the cell. Hit "Enter" when you made your change or click anywhere else on the screen. The new new value will be saved immediately.

You can cancel editing by simply pressing "Esc" on your keyboard. This will revert the change you already made in the editor and bring back the old value.

Move a transaction

It is possible to change the order of transactions in the table. Right-click the row and select "Move Up" or "Move Down" from the context menu. The same action can be triggered by pressing Ctrl+Up or Ctrl+Down respectively.

Convert transactions to budgets

Transactions can be converted into budgets and vice versa. Right-click on a row and select "Convert to Budget" or "Convert to Transaction" respectively.

Delete a transaction

Of course, you can delete transactions and budgets from the transaction table:

  • Right-click on the transaction table. Select "Delete" from the context menu.
  • Click the "Delete" button directly above the transaction table.
  • Click the "Delete" button from the view's toolbar.
  • Press the "Delete" key.

RsBudget will ask you for confirmation before actually deleting the row(s) selected.

Export the plan

RsBudget can export your plan to a comma-separated file (CSV) or an Excel sheet. Click the "Export" button in the general toolbar or select the item from the "File" menu. Select the target file of you export in the appearing dialog and click "OK".
