Categories

This view provides the managent of transaction categories.

Data Description

The view consists of a single table

Column Description
Default Whether the category is the default category for transactions. Only one category can have the flag set.
Name Name of category

Tasks

Create a new category

Create a new category by

  • Right-click on the categories table. Select "New Category" from the context menu.
  • Click the "New Category" button from the view's toolbar.
  • Press the "Insert" key.

All methods will create a new category at the end of the table and scroll to the new row in order to make it visible. The editing cursor is placed within the Name field so you can start typing.

Modify a category

Double-click the value in the table that you want to modify. Depending on the column, the checkbox will be ticked or a text editor will appear so you can change the current value of the cell. Hit "Enter" when you made your change or click anywhere else on the screen. The new new value will be saved immediately.

You can cancel editing by simply pressing "Esc" on your keyboard. This will revert the change you already made in the editor and bring back the old value.

Delete a catgeory

Of course, you can delete categories from the table:

  • Right-click on the table. Select "Delete" from the context menu.
  • Click the "Delete" button from the view's toolbar.
  • Press the "Delete" key.

RsBudget will ask you for confirmation before actually deleting the row(s) selected. The row will be made invisible only when there are transactions assigned to the category.
