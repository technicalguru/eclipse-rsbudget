History

This view displays the historical records of balances. This includes all accounts and any additional historical items that you defined in the settings dialog.

Data Description

The table contains a row for each month in history. The columns list the individual values recorded. Turquoise columns are calculated by RsBudget and cannot be edited.

Column Description
Date The date in history. Bold text marks the beginning of a new year.
Status The recorded Opening Balance of the date.
Change The difference of the value in comparison to the row above.
Car Total Miles
(Example)		 An additional value as defined by the settings dialog.

The Settings Dialog defines whether the "Change" column is displayed for additional values.

Tasks

Modify historical data

Turquoise columns are calculated by RsBudget and cannot be edited. All other values can be edited by double-clicking the cell. An editor will appear allowing you to enter, modify or delete the value. Hit "Enter" to finish the input or press "Esc" to cancel editing.
