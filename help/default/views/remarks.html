Remarks

The view displays an editable remark attached to the selected month in the Monthly Budget Planning view.

Data Description

Only the remark as entered will be displayed. An asterisk is displayed next to the view's title as long as the remark is not yet saved in the database.

Tasks

Modify the remark

You can modify the remark at any time. Click into the text area in order to place the editing cursor. Enter the text that you want to store along with the current month. The remark will be saved after 5 seconds of inactivity. The view's title indicates the persistence status with an asterisk as long as the remark is not yet saved.
