General Settings

Language

This combo box control which language all texts (and this help) will be displayed with. Although there are currently only a few languages implemented, you should select your personal language nevertheless. The system will automatically use English when a text is not available in your language.

The language settings also controls how amounts are displayed. E.g. The US-style of displaying amounts is to prefix the value with the currency symbol. Negative values appear in brackets, e.g. "(USD2,000.00)". As you can see from the example, digits are grouped with a comma, the decimal separator is the dot. The German display would be "-2.000,00 EUR". Negative amounts are also colored red by default. This setting can be controlled, too.

A future version will add special settings for the format. However, current version depends on your language settings.

This setting takes effect completely once you restarted the application.

Currency

This setting controls the currency symbol that will be displayed. The default value is the currency derived from your language setting.

This setting takes effect completely once you restarted the application.

Reset Layout Changes

The application has a default layout for its size, position and order of views. As RsBudget remembers your personal changes, there could be the situation that you are uncomfortable with the changes you made before. Tick the box and the application will reset all your layout changes at next startup.

A layout reset also takes place when you update the application.

Restart Application

This link allows you to immediately save your changes in the Settings Dialog and restart the application. A confirmation dialog will ensure that the restart does not occur because you clicked the link by accident.
