Statistik

RsBudget hält alle Daten in einer Datenbank, die eine spätere statistische Analyse und Recherche erlaubt. Die aktuelle Version enthält noch keine Funktionen für die statistische Analyse, bietet aber eine einfache Möglichkeit, die bisherigen Salden abzurufen und so einen Überblick über den bisherigen finanziellen Status zu bekommen. Die Historie liefert den Eröffnungs-Saldo eines jeden Planungsmonats. Zusätzlich können weitere, beliebige Werte je Monat festgehalten werden, z.B. der Kilometerstand eines Autos oder der Wert im Aktiendepot.

Diese zusätzlichen Werte werden im Einstellungs-Dialog definiert. Die Historie listet dann all diese Werte je Monat auf. Monetäre Werte werden mit ihrem Vormonat verglichen, die frei definierten Werte nur dann, wenn dies im Einstellungs-Dialog so festgelegt wurde.
