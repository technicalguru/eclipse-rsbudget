Statistics

RsBudget keeps all data in a database which allows later statistical analysis and historical research. The current version does not yet include statistical analysis but provides a simple historical overview of your financial status. The History View contains the status as of the 1st of each month of your planning. Additional values can be stored separately such as your car mileage or the value of your shares.

These additional values must be defined within the Preference Dialog. The view will list all your important data per month. Monetary values will also list the difference from the previous month. Custom values have this difference listed only when defined in the Preference Dialog.
