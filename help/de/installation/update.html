Online Update

RsBudget kann online auf die jeweils neueste Version aktualisiert werden. Dazu benötigen Sie lediglich eine funktionierende Internet-Verbindung. Da es keine automatische Benachrichtigung gibt, müssen sie die Aktualisierung manuell anstoßen. Das Programm wird die Update-Seite kontaktieren. Falls eine neue Version vorliegt, werden Sie um Erlaubnis gebeten, diese herunterzuladen und zu installieren. Wenn die Aktualisierung abgeschlossen ist, müssen Sie das Programm erneut starten, damit die neue Version aktiviert wird.

Sie können die Verfügbarkeit einer neuen Version auch feststellen, wenn Sie die RsBudget Homepage besuchen.
