RsBudget installieren

Laden Sie RsBudget von der Produkt-Homepage herunter. Achten Sie dabei auf die richtige Plattform für Ihren PC.

Hinweise für Windows

Windows-Benutzer entpacken das Archiv auf Ihre Festplatte, vorzugsweise in das Verzeichnis C:\Program Files.

Hinweise für Linux

Bitte entpacken Sie das tar.gz-Archiv in ein geeignetes Verzeichnis, z.B. /usr/local.

Hinweise für Mac

Bitte entpacken Sie das Archiv in ein geeignetes Verzeichnis, von wo aus Sie das Programm starten können.
