Voraussetzungen

Bevor Sie RsBudget installieren können, müssen Sie weitere, zusätzliche Software installieren. Dieser Abschnitt beschreibt, wie Sie Ihren PC vorbereiten.

Java Runtime Environment

RsBudget basiert auf der Eclipse Rich Client Platform (RCP), die wiederum auf Java basiert. Java ist jedoch nicht Bestandteil des Installationspakets von RsBudget. Deshalb müssen Sie eine aktuelle Java-Laufzeitumgebung (JRE) selbst installieren. Bitte installieren Sie mindestens Java 7 auf Ihrem Computer. RsBudget wurde mit Oracle Java entwickelt und getestet. Daher ist es empfehlenswert, dieses Produkt zu verwenden. Befolgen Sie die jeweiligen Installationsanweisungen Ihrer Java-JRE.

Datenbank-Software

RsBudget speichert sämtliche Daten in einer SQL-Datenbank. Hier ist eine Auswahl:

  • HyperSQL (HSQLDB) - keine weitere Installation erforderlich
  • MySQL 5.x
  • Oracle - unterstützt, jedoch nicht getestet (siehe auch Andere Datenbank-Software)
  • DB2 - unterstützt, jedoch nicht getestet (siehe auch Andere Datenbank-Software)

Für HyperSQL wird keine separate Installation benötigt. Die Software ist bereits im RsBudget-Installationspaket enthalten.

Verwendung von HyperSQL

Für HyperSQL müssen keine weiteren Installationsschritte durchgeführt werden. Die erforderlichen Bibliotheken sind bereits im RsBudget-Installationspaket enthalten. HyperSQL ist nur die zweitbeste Wahl , da es unzureichend mit RsBudget getestet wurde.

Verwendung von MySQL

MySQL 5.x sollten Sie nur dann verwenden, wenn Sie sich mit dieser Software auskennen. Installieren Sie die Software anhand der MySQL-Installationsanweisungen. Außerdem müssen folgende Schritte durchgeführt werden:

  1. Erstellen Sie eine Datenbank, z.B. mit Namen rsbudget.
  2. Erstellen Sie einen Datenbanknutzer rsbudget (andere Namen erlaubt) und vergeben Sie ein Passwort.
  3. Berechtigen Sie den neuen Benutzer, alle Operationen auf der neuen Datenbank auszuführen. Dies sollte insbesondere die CREATE-, ALTER- und DROP-TABLE-Berechtigungen sowie die SELECT-, INSERT-, UPDATE-, DELETE-Berechtigungen beinhalten.

Notieren Sie sich die gemachten Einstellungen für MySQL, da RsBudget während der initialen Einrichtung danach fragen wird.

Andere Datenbank-Software

Prinzipiell ist es möglich, auch jede andere Datenbank-Software zu verwenden, die JDBC und Hibernate unterstützt. Sollten Sie sich für eine solche Software entscheiden, müssen Sie einige weitere Installationsschritte ausführen. Bitte beachten Sie aber, das die aktuelle RsBudget-Version diesbezüglich nicht getestet wurde und es zu Fehlern kommen kann. Daher kann dies nur erfahrenen Java-Entwickler empfohlen werden. Befolgen Sie bitte folgende Anweisungen:

  • Installieren Sie die Datenbank-Software
  • Erstellen Sie einen Datenbankbenutzer
  • Erstellen Sie eine Datenbank-Instanz, die die RsBudget-Daten aufnehmen soll
  • Erzeugen Sie die Datenbank-Struktur
  • Erzeugen Sie die Basis-Daten innerhalb der Datenbank
  • Erstellen Sie die Datei dbconfig.xml in Ihrem Home-Verzeichnis (Unterverzeichnis .rsbudget)
  • Stellen Sie den richtigen JDBC-Treiber per Classpath für RsBudget zur Verfügung

Die beste Methode, diese Konfiguration vorzunehmen, ist es, RsBudget zunächst fü MySQL einzurichten und danach alle MySQL-Daten in Ihre gewünschte Datenbank zu kopieren. Anschließend müssen Sie lediglich die Datei dbconfig.xml anpassen, die RsBudget während des Setups im Verzeichnis $HOME/.rsbudget angelegt hat.

Zukünftige RsBudget-Versionen werden weitere Datenbank-Produkte unterstützen.
