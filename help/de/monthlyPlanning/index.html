Expense Tracking

You are ready to track and control your expenses as soon as you created your budget plan. Tracking is what you will do most of the time. Entering new transactions, changing planned amounts and controlling your financial activities.

You can change any transaction or budget without affecting other months or the template. The changes are kept within the respective month only. This article explains how to properly track your expenses.

A complete description of the view can be found here.

Creating the plan

Navigate to the respective month (the current month is opened by default when you start RsBudget). Create the plan by using the button from the toolbar. This action will evaluate your General Budget Plan and create all transactions and budgets that are planned for the given month. The Summary Section will now tell you the basic figures if your actual transactions will occur as planned.

Opening the month

The next step is to tell RsBudget what the opening status of your accounts was. This value is defined individually for each month. So it is not required that is equal to the closing status of the previous months. However, it is recommended to have them aligned. The opening status is also the value that occurs in the History View.

You set the status by clicking the button from the the view's toolbar. The dialog that appears will evaluate all possible options and propose a few possible amounts. You either select one of them or enter the amount manually. Please make sure that you entered the amount of the left side of the dialog, as the right side is reserved for the closing status.

Tracking actual transactions

You can now start tracking income and expenses. Create transactions or budgets, enter actual amounts and their transaction dates throughout the month. The system will reflect your changes in the Summary Section and recalculate Profit/Loss values according to existing and remaining transactions. Details about the various actions are described in the view's help page.

It is possible to simplify the assignment of categories and budgets by using respective rules. Please refer to the Preferences Dialog description for further information.

Closing the month

Finally, you will close your month financially when the last transaction has been entered. The last step is therefore to define the closing status of your accounts. This value defines the Profit/Loss of the month. It is saved separately so there is actually no requirement that it fits logically to your transactions of the month. But it makes more sense to take over the proposal from RsBudget when assigning the closing status. click the button again and set the value at the right side of the dialog.

That's it. You successfully tracked and controlled your finances of this month and can start over again with the next month.
