Allgemeine Planung

Die Allgemeine Planung legt fest, welche Transaktionen und Budgets in den einzelnen Monaten eines Jahres geplant werden. Sie können sich diese Planung als eine Art Vorlage vorstellen. Diese Vorlage wird dann für die Planung jedes einzelnen Monats verwendet. Jeder Monat stellt also eine konkrete Instanz dieser Vorlage dar. Die Ein- und Ausgabenkontrolle geschieht dann anhand dieser konkreten Instanz.

Die Planungsvorlage kann jederzeit geändert werden. Die bereits existierenden Monatspläne werden dadurch nicht beeinflusst. Die Vorlage wird nur zum Zeitpunkt der Erstellung des Monatsplans verwendet.

Eine vollständige Beschreibung der Ansicht findet sich hier.

Eine Transaktion anlegen

Drücken Sie die Einfügen-Taste, um eine neue Zeile zu erstellen. Das System wird die neue Zeile am Ende der Tabelle hinzufügen und ggf. die Ansicht dorthin scrollen. Der Cursor ist in der Text-Spalte der neuen Transaktion platziert. Sie können sofort mit dem Eingeben eines Namens beginnen. Drücken Sie die Tab-Taste oder doppelklicken Sie auf das Soll-Feld, um den entsprechenden Planwert der Transaktion anzugeben. Ausgaben werden durch negative Zahlen repräsentiert, Einnahmen durch positive Werte. Geben Sie also "-10" ein, wenn Sie Ausgaben in Höhe von 10 Euro angeben wollen.

Das soeben beschriebene Szenario hat nun eine monatlich auftretende Transaktion erzeugt. Das heißt, dass RsBudget eine entsprechende Transaktion in jedem einzelnen Monatsplan erzeugen wird.

Die Periodizität ändern

Einige Transaktionen werden jedoch nicht jeden Monat benötigt, sondern nur vierteljährlich oder einmal im Jahr. Sie können diese Eigenschaft festlegen, wenn Sie auf die Zyklus-Spalte doppelklicken. Diese enthält als Voreinstellung den Wert "monatlich". Ändern Sie den Zyklus wie gewünscht ab.

Transaktionen, die nicht monatlich auftreten, benötigen eine weitere Informationen: nämlich, in welchen Monaten sie geplant werden sollen. Je nach Zyklus können die Optionen variieren. Die aktuelle "Sequenz" ist in der entsprechenden Spalte aufgeführt. Doppelklicken Sie dieses Feld, um den Wert abzuändern.

Eine Kategorie zuweisen

Neue Transaktionen werden automatisch der Standard-Kategorie zugewiesen. Ändern Sie die Kategorie ab, indem Sie auf das entsprechende Feld doppelklicken und die Kategorie auswählen. Die Auswahlmöglichkeiten selbst können in der Ansicht Kategorien verändert werden.

Ein Budget anlegen

Ein Budget ist in der RsBudget-Terminologie eine Gruppe gleichartiger Transaktionen. Es wird benutzt, wenn man nicht genau planen kann, wieviele solcher Transaktionen und in welcher Gesamthöhe auftreten werden. "Lebensmittel" ist ein Beispiel dafür. Sie gehen vermutlich mehrmals im Monat einkaufen, möchten aber insgesamt nicht mehr als 100 Euro dafür ausgeben. Anstatt mehrere Transaktionen auf Verdacht anzulegen, benutzen Sie ein Budget.

Ein Budget wird aus einer bereits existierenden Transaktion erzeugt. Der erste Schritt ist also, solch eine Transaktion zu erzeugen (siehe oben). Als Nächstes klicken Sie mit der rechten Maustaste auf diese Transaktion. Aus dem erscheinenden Kontextmenü wählen Sie "In Budget umwandeln" aus. RsBudget konvertiert die Transaktion nun in ein Budget und verschiebt es an den Anfang der Tabelle (alle Budgets erscheinen ganz oben). Außerdem wird das Budget mit türkis als Hintergrundfarbe gekennzeichnet, so dass Sie Budgets schnell von gewöhnlichen Transaktionen unterscheiden können.

Ein Budget zuweisen

Obwohl es nicht sehr wahrscheinlich ist, dass vereinzelte Transaktionen einem Budget bereits in der Allgemeinen Planung zugeordnet werden, kann man solche Transaktionen innerhalb von Budgets planen. Doppelklicken Sie auf die Budget-Spalte der entsprechenden Transaktion und wählen Sie das zugehörige Budget aus.

Die Reihenfolge ändern

Die Reihenfolge der Budgets und Transaktionen in der Vorlage kann über das Kontext-Menü (Rechtsklick mit der Maus) oder durch Drücken der Strg+↑ und Strg+↓ Tasten verändert werden. Diese Reihenfolge wird benutzt, wenn RsBudget den konkreten Monatsplan erzeugt. Auf bereits existierende Monatspläne hat auch diese Änderung (wie alle anderen hier beschriebenen Aktionen) keinen Einfluss.

Eine Transaktion löschen

Eine Transaktion oder ein Budget wird gelöscht, indem man die jeweilige(n) Zeile(n) zunächst selektiert. Danach kann die selektierte Zeile durch Drücken der Entfernen-Taste oder die Schaltfläche gelöscht werden. Bestätigen Sie den erscheinenden Dialog und die Zeile wird aus der Tabelle entfernt.

Auf bereits existierende Monatspläne hat diese Änderung keinen Einfluss.
