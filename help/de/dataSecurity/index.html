Datensicherheit und Datenschutz

RsBudget schützt und respektiert die Privatsphäre Ihrer Daten. Deshalb werden keinerlei persönliche Daten von Ihrem Rechner übertragen. Die Software-Aktualisierung erfolgt anonym und wird nicht auotmatisiert durchgeführt. Auf diese Weise kontrollieren Sie selbst, wann RsBudget den Update-Server kontaktieren soll. Ebenso werden dabei keine persönlichen Daten übertragen.

Die Anwendung selbst ist durch ein persönliches Passwort geschützt. Dieses wird nirgends auf Ihrer Computer gespeichert. Das Passwort können Sie jederzeit ändern.

Ihre persönlichen Daten weren in einer Datenbank auf Ihrem Rechner gespeichert (es sei denn, Sie haben eine entfernte Datenbank konfiguriert). Sie können selbst bestimmen, ob diese Daten verschlüsselt abgelegt werden oder nicht. Solange niemand Ihre Datenbank-Zugangsdaten stiehlt oder anderweitig Zugang zur Datenbank erlangt, sind Ihre Daten dort sicher und können nicht ausgelesen werden.

Sie sollten sich jedoch darüber bewusst sein, dass Ihre Daten - wie bei jeglicher Software - nur so lange sicher sind, so lange Sie Ihren Computer entsprechend schützen. Daher hier einige Hinweise, wie Sie Ihren Computer und Ihre Daten schützen können:

  • Seien Sie vorsichtig, wenn Sie unbekannte Webseiten besuchen, da diese Ihren PC mit Malware oder einem Virus infizieren können. Antivirensoftware gewährleistet keinen 100-prozentigen Schutz, denn:
    • Sie könnte die Schadsoftware nicht kennen, mit der Ihr PC infiziert ist.
    • Sie könnte nicht auf dem aktuellsten Stand sein.
    • Sie wiegt Sie in falscher Sicherheit und macht Sie unvorsichtig.
  • Deaktivieren Sie die automatische Anzeige von Bildern in Ihrem E-Mail-Programm.
  • Klicken Sie niemals auf Links in E-Mails von unbekannten Absendern (oder selbst von bekannten Absendern), es sei denn, Sie wissen genau, was Sie tun.
  • Seien Sie vorsichtig, wenn Sie fremde USB-Sticks oder -Laufwerke benutzen wollen. Deaktivieren Sie die Auto-Start-Funktion an Ihrem Rechner.
  • Benutzen Sie Passwörter nur einmal. Angreifer können so ein einmal gestohlenes Passwort nicht an anderer Stelle verwenden. (Generieren Sie Passwörter mit einem einfach zu merkenden, aber schwer zu erratenden Muster).
  • Seien Sie beim Eingeben von Passwörtern vorsichtig, dass Sie niemand dabei beobachtet.
  • Installieren Sie keine unbekannte Software. Recherchieren Sie die Seriösität der Software, bevor Sie sie installieren.
  • Schreiben Sie Ihre Passwörter niemals auf, weder auf Papier noch in elektronische Dateien.
