Erste Schritte

Die Benutzeroberfläche

RsBudget basiert auf Eclipse RCP, das schon viele benutzerfreundliche Features mitbringt. Das Haupt-Fenster ist in zwei Bereiche aufgeteilt. Der Hauptbereich enthält die Allgemeine Planung und die Monatliche Ausgabenkontrolle. Außerdem finden Sie hier die historischen und statistischen Werte Ihrer Finanzaktivitäten. Alle drei Features sind in einem "Tab-Panel" zusammengefasst. Selektieren Sie ein "Tab", indem Sie auf den entsprechenden Titel klicken.

Auf der rechten Seite finden Sie zusätzliche Ansichten, wie die für die Kategorien, die Detailsansicht und eine spezielle Maske zur Erfassung von Notizen für den jeweiligen Monat. Die Größe dieser Ansichten kann durch einfaches "Ziehen" der Ränder verändert werden. Klicken Sie dazu mit der Maus auf den Rand, halten Sie die Maustaste gedrückt und bewegen Sie die Maus dann in die entsprechende Richtung. Sobald Sie die gewünschte Position erreicht haben, lassen Sie die Maustaste wieder los. RsBudget speichert Ihre Änderung ab und positioniert die Ansichten entsprechend Ihren Wünschen auch beim nächsten Start.

Ein weiteres, nützliches Features ist das Verändern der Anordnung der Ansichten. So können Sie zum Beispiel die Detailansicht in der rechten, unteren Ecke platzieren. Klicken Sie dazu auf den Titel der Detailansicht und halten Sie die Maustaste gedrückt. Ziehen Sie nun die Ansicht an die neue Position. Farbige Linien helfen Ihnen dabei, vorherzusehen, wo die Ansicht bei Loslassen der Maustaste landen würde. Auf diese Weise können Sie Ansichten sogar vom Hauptfenster "abkoppeln". Ziehen Sie dazu die Ansicht einfach außerhalb des Hauptfensters, z.B. auf einen zweiten Bildschirm.

Alle Änderungen können zurückgesetzt werden, in dem Sie im Einstellungs-Dialog den Haken bei "Window-Layout beim nächsten Start zurücksetzen" machen. RsBudget wird dann bei erneutem Start Ihre Änderungen löschen.

Jedes "Tab" besitzt eine Werkzeugleiste (Toolbar), die rechts vom Titel angezeigt wird, wenn Sie den "Tab" selektiert haben. Diese Toolbar stellt Schaltflächen zum Auslösen diverser Aktionen bereit, zum Beispiel zum Einfügen oder Löschen von Tabellenzeilen, dem Sortieren von Tabellen oder andere sinnvolle Aktionen. Eine Schaltfläche ist grau dargestellt, wenn die entsprechende Aktion derzeit nicht ausgelöst werden kann. So ist zum Beispiel die Schaltfläche für das Löschen von Tabellenzeilen deaktiviert, wenn keine Zeile zum Löschen ausgewählt wurde.

Das Hauptfenster besitzt außerdem ein Hauptmenü und eine globale Werkzeugleiste. Dort sind die wichtigsten Aktionen für Sie hinterlegt. Hier können Sie RsBudget beenden oder den Einstellungs-Dialog öffnen. Das Hilfe-Menü stellt Dokumentation und Hilfe-Texte bereit.

So geht's los

Schritt 1 - Kategorien anlegen

Sie sollten zunächst mit dem Anlegen von Kategorien beginnen. Sie finden die Ansicht rechts oben im Hauptfenster. Zu Anfang enthält die Liste nur eine einzelne Kategorie. Diese haben Sie angelegt, als Sie RsBudget konfiguriert haben. Kategorien gruppieren Ihre Transaktionen in verschiedene Bereiche und dienen daher zur logischen Zusammenfassung von Transaktionen. Typische Kategorien sind zum Beispiel "Lebensmittel", "Energiekosten" oder "Versicherungen". In der Statistik-Ansicht können Sie später diese Kategorien anzeigen lassen, um Summen oder Trends über eine Zeitspanne anzuzeigen.

Es ist nicht zwingend, viele Kategorien anzulegen. Sie können diese auch später verändern. Allerdings erhöht sich Ihr Aufwand für die Zuordnung von neuen Kategorien, je mehr Transaktionen bereits erfasst worden sind. Legen Sie daher zumindest ein für Sie sinnvolles Minimal-Set an Kategorien an.

Selektieren Sie die Kategorien-Ansicht und drücken Sie entweder die Einfügen-Taste oder klicken Sie auf die Schaltfläche . Am Ende der Tabelle wird eine neue Zeile erscheinen, der Cursor ist bereits zur Eingabe eines neuen Namens positioniert. Geben Sie den neuen Titel ein und drücken Sie die Enter-Taste.

Eine Kategorie ist als "Standard" markiert. Alle Transaktionen, die keiner anderen Transaktionen zugeordnet sind, werden automatisch dieser Kategorie zugewiesen.

Eine "Regel-Maschine" erleichtert Ihnen später die Zuweisung von Kategorien. Weitere Informationen finden Sie hier.

Schritt 2 - Den Allgemeinen Plan anlegen

Sie benötigen eine allgemeine Budgetplanung, die beschreibt, wie die einzelnen Monate Ihrer Finanzplanung aussehen werden. Selektieren Sie dazu die Allgemeine Planung und fügen Sie eine neue Transaktion ein. Dies geschieht auf dieselbe Weise, wie Sie es schon für Kategorien getan haben. Drücken Sie die Einfügen-Taste oder klicken Sie auf die Schaltfläche .

Der Allgemeine Plan ist eine Art Muster. Er stellt noch keine realen Transaktionen dar und dient so lediglich zur Planung Ihrer Finanzen. Erzeugen Sie Transaktionen für die monatliche Miete, Versicherungsbeiträge, Gehaltszahlungen und was immer Sie an regelmäßigen Zahlungen vorhersehen können.

Nicht alle Transaktionen werden monatlich ausgeführt. Einige von ihnen werden nur alle zwei, drei oder sogar zwölf Monate auftreten. Die Spalten Zyklus und Sequenz steuern dieses Verhalten.

Schließlich wandeln Sie einzelne Transaktionen noch in Budgets um. Budgets sind virtuelle Brieftaschen, aus denen Sie bestimmte, konkrete Zahlungen tätigen wollen, z.B. "Benzin".

Schritt 3 - Einen Monatsplan anlegen

Der Allgemeine Plan ist die Vorlage, um konkrete Monatspläne erzeugen zu können. Selektieren Sie die Monatsplanung und navigieren Sie zum jeweiligen Monat (der aktuelle Monat ist bereits voreingestellt). Klicken Sie die Schaltfläche . Dies wird den Monatsplan anhand der Vorlage des Allgemeinen Plans erzeugen.

Der Monatsplan enthält alle realen Transaktionen, die in dem jeweiligen Monat auftreten werden. Die Soll-Spalte enthält den Wert, den Sie zuvor im Allgemeinen Plan festgelegt haben. Eine weitere Spalte (Ist) wird den jeweils wirklichen Betrag enthalten, der transferiert wurde. Die erste Spalte (Datum) enthält dann das Wertstellungsdatum.

Sie können Transaktionen oder Budgets nach Belieben hinzufügen, verändern oder löschen, wie Sie es auch schon zuvor in der Allgemeinen Planung getan haben.

Schritt 4 - Einnahmen und Ausgaben kontrollieren

Um eine reale Transaktion einzugeben, suchen Sie die entsprechende Zeile und doppelklicken Sie auf das leere "Ist"-Feld. Geben Sie den tatsächlichen Wert ein. Ausgaben müssen mit einem vorangestellten Minus ("-") gekennzeichnet werden, Einnahmen sind durch positive Werte bestimmt. Drücken Sie die Enter-Taste, um die Eingabe abzuschließen. RsBudget setzt das Transaktionsdatum dann automatisch auf den aktuellen Tag.

Alternativ können Sie auch auf das leere "Datum"-Feld der entsprechenden Zeile doppelklicken und das Datum entsprechend auswählen. RsBudget setzt den Ist-Wert dann automatisch auf den eingetragenen Soll-Wert. Doch ganz gleich, welche Methode Sie benutzen, Sie können alle Werte jederzeit durch Doppelklick auf das jeweilige Feld verändern.

Falls Sie eine ungeplante Transaktion eingeben müssen, fügen Sie einfach eine neue Zeile durch Drücken der Einfügen-Taste oder durch Klick auf die Schaltfläche hinzu. Obsolete bzw. überflüssige Transaktionen können durch Selektieren und Drücken der EntfernenTaste bzw. Klicken auf die Schaltfläche entfernt werden. Auf dieselbe Weise können Sie übrigens auch eine reale Transaktion wieder entfernen, so dass nur die geplante Transaktion stehen bleibt.

Alle Änderungen, die Sie vornehmen, werden sofort in der Datenbank gespeichert. Daher ist es nicht erforderlich, explizit alles abzuspeichern. Ebenso werden die Statistik-Werte über der Tabelle automatisch angepasst, wenn Sie Änderungen vorgenommen haben. Diese Statistikwerte geben Auskunft über Soll- und Ist-Werte und prognostizieren die Gewinn-/Verlustrechnung für den laufenden Monat.

Schritt 5 - Die Statistik füttern

Nach Abschluss des Monats müssen Sie noch die Statistik des Systems "füttern", indem Sie die Salden zu Beginn und Ende des jeweiligen Monats festhalten.Klicken Sie auf die Schaltfläche und geben Sie Ihre Werte entsprechend ein. Der Dialog schlägt Ihnen einige sinnvolle Werte vor, so dass Sie die Eingabe leichter vornehmen können.

Wie es weitergeht
