Allgemeine Einstellungen

Sprache

Diese Auswahlbox bestimmt, in welcher Sprache alle Texte (und die Hilfe) angezeigt werden. Obwohl derzeit nur wenige Sprachen implementiert sind, sollten Sie Ihre gewünschte Sprache trotzdem auswählen. Das System wird dann Englisch verwenden, solange die Sprache nicht verfügbar ist.

Die Spracheinstellung steuert auch, wie Geldwerte angezeigt werden. So wird das Währungssymbol in der US-Sprachfassung immer vorangestellt. Ausgaben erscheinen in Klammern, z.B. (USD2,000.00). Wie aus dem Beispiel ersichtlich, werden Tausender- Stellen durch Kommantas gruppiert, der Dezimaltrenner ist der Punkt. Die entsprechende deutsche Anzeige wäre -2.000,00 EUR. Ausgaben werden außerdem farblich in rot dargestellt. Diese Einstellung kann ebenfalls verändert werden.

Eine zukünftige Erweiterung wird es erlauben, dieses Format genauer zu kontrollieren. Gegenwärtig wird das Format jedoch durch die gewählte Sprache bestimmt.

Diese Einstellung wird erst wirksam, wenn Sie die Anwendung neu starten.

Währung

Diese Einstellung legt fest, welches Währungssymbol verwendet wird. Der Eintrag wird initial durch Ihre Sprache bestimmt.

Diese Einstellung wird erst wirksam, wenn Sie die Anwendung neu starten.

Layout zurücksetzen

Die Anwendung hat ein vordefiniertes Layout bzgl. Größe, Position und Reihenfolge der Anzeige-Tabs. Da RsBudget Ihre persönlichen Änderungen an diesem Layout speichert und bei Neustart wiederherstellt, kann es vorkommen, dass Sie mit Ihren Änderungen unzufrieden sind. Klicken Sie die Box an, um beim nächsten Start das vordefinierte Layout zurückzuholen.

Das Zurücksetzen wird erst wirksam, wenn Sie die Anwendung neu starten.

Anwendung neu starten

Dieser Link erlaubt es Ihnen, Ihre Einstellungen sofort abzuspeichern und einen Neustart der Anwendung durchzuführen. Ein Bestätigungsdialog stellt sicher, dass Sie den Neustart nicht versehentlich durch Klicken auf den Link auslösen.
