Netzwerk

Dieser Reiter legt fest, auf welche Weise sich RsBudget mit dem Internet verbinden soll. Die Einstellungen sind insbesondere für das Online-Update und externe Links in der Hilfe wichtig.

Drei Auswahlmöglichkeiten stehen zur Verfügung:

  • Kein Proxy: Sie benötigen keinen Proxy. Ihr Computer hat direkte Verbindung zum Internet.
  • Systemeinstellungen verwenden: Die Einstellungen werden von Ihrem Internet-Browser bzw. System übernommen. Dies ist die Voreinstellung. Wählen Sie diese Option, wenn Sie unsicher sind.
  • Manuell: Sollten Sie die Proxy-Einstellungen genau kennen, setzen Sie hier Server und Port Ihres Proxies sowie ggf. die Authentifizierungsdaten.

Bitte beachten Sie, dass unter Umständen diese Einstellungen nicht sofort wirksam werden, sondern erst nach einem Neustart der Anwendung. Sollten Sie Probleme mit dem Online Update haben, ändern Sie die Proxy-Einstellungen oder starten Sie die Anwendung erneut, damit die Einstellungen aktiv werden.
