Network Settings

This tab controls how RsBudget connects to the internet. The setting is required for the Online Update and when you click external links in the Online Help Browser.

Three options are available:

  • No Proxy: You don't require any proxy settings. Your computer has a direct internet connection.
  • Use System Settings: The settings from your system's Internet Browser will be taken. (Use this when in doubt, default setting also)
  • Manual: When you know exactly your proxy settings. Set the proxy server and port and, if required, any proxy authentication.

Please notice that sometimes these settings are not effective until the next start of the application. If you experience problems with the Online Update, either change the settings or try to restart the application to ensure the settings are applied.
