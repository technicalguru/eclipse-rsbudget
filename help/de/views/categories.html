Kategorien

Diese Ansicht wird zum Verwalten der verfügbaren Kategorien verwendet.

View-Beschreibung

Die Ansicht besteht aus einer einzelnen Tabelle:

Spalte Beschreibung
Default Gesetzt, wenn die Spalte standardmäß für Buchungen verwendet wird. Nur eine Kategorie kann dieses Flag gesetzt haben.
Name Name der Kategorie

Aufgaben

Eine neue Kategorie anlegen

Erstellen Sie eine neue Kategorie, indem Sie

  • auf die Tabelle mit der rechten Maustaste klicken. Wählen Sie "Neue Kategorie" aus dem Kontextmenü.
  • Auf die Schaltfläche aus der Werkzeugleiste der Ansicht klicken.
  • Die Einfügen-Taste drücken.

Alle drei Methoden erzeugen eine neue Kategorie am Ende der Tabelle. Die Textmarke (Cursor) ist bereits im Namensfeld platziert, so dass Sie nur noch den Namen eingeben müssen.

Eine Kategorie bearbeiten

Doppelklicken Sie in der Tabelle auf den Wert, den Sie verändern wollen. Je nach Spalte wird entweder die Checkbox angewählt oder ein Textfeld erscheint, damit Sie den Wert verändern können. Drücken Sie Enter wenn Sie Ihre Eingabe beendet haben oder klicken Sie mit der Maus irgendwo außerhalb des Textfeldes. Die Änderung wird sofort gespeichert.

Sie können das Editieren abbrechen, wenn Sie Esc auf Ihrer Tastatur drücken. Ihr Änderung wird dann nicht abgespeichert.

Eine Kategory löschen

Natürlich können Sie Kategorien auch wieder löschen:

  • Klicken Sie mit der rechten Maustaste auf die Tabelle. Wählen Sie "Löschen" aus dem Kontextmenü.
  • Klicken Sie auf die Schaltfläche aus der Werkzeugleiste.
  • Drücken Sie die Entfernen-Taste.

RsBudget wird Sie zur Sicherheit noch einmal um Rückbestätigung bitten, bevor die ausgewählte Zeile gelöscht wird. Die Kategorie wird jedoch nur versteckt, wenn es Buchungen gibt, die mit dieser Kategorie verknüpft sind.
